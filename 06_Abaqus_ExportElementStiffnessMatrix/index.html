Abaqus输出单元刚度矩阵

关键字

*ELEMENT MATRIX OUTPUT

此keyword用于将元素刚度矩阵和质量矩阵(如果可用)写入结果文件、用户定义文件或数据文件.只对Abaqus/Standard求解器有效 ;属于历程输出;

必须的参数

  • ELSET=[elset_name]

用于定义需要输出的element set

可选参数

  • STIFFNESS
  • OUTPUT FILE
  • MASS
  • FREQUENCY
  • FILE NAME
  • DLOAD

FREQUENCY :指定输出频率(unit:increment)

OUTPUT FILE、FILE NAME搭配使用可以指定输出文件格式

使用关键字

*ELEMENT MATRIX OUTPUT,ELSET=elset_name,DLOAD=YES,STIFFNESS=YES,MASS=YES

输出单元集elset_name的MASS Matrix/DLOAD Matrix/STIFFNESS Matrix(IF it is available for the analysis step);数据会输出到result file(因为没修改OUTPUT FILE参数);

输出到mtx文件

*ELEMENT MATRIX OUTPUT,ELSET=elset_name,DLOAD=YES,STIFFNESS=YES,MASS=YES,OUTPUT FILE=USER DEFINED,FILE NAME=UserDefineName

输出单元集elset_name的MASS Matrix/DLOAD Matrix/STIFFNESS Matrix(IF it is available for the analysis step);数据会输出到UserDefineName.mtx文件

输出到dat文件

*ELEMENT MATRIX OUTPUT,ELSET=elset_name,DLOAD=YES,STIFFNESS=YES,MASS=YES,OUTPUT FILE=USER DEFINED

输出单元集elset_name的MASS Matrix/DLOAD Matrix/STIFFNESS Matrix(IF it is available for the analysis step);数据会输出到_JobName.dat文件
